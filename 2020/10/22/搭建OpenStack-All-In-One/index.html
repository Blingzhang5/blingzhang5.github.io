搭建OpenStack-All In One

  • OpenStack

ALL IN ONE 版本的openstack

准备一台虚拟机

1.配置好虚拟机的网卡

2.配置好两张网卡

3.使网络可以Ping通网络

4.安装脚本包

yum install iaas-xiandian -y

4.1查看脚本包

ll /usr/local/bin/

5.编辑openrc.sh文件

6.安装qpid服务

7.通过脚本安装 mysql 服务

iaas-install-mysql.sh

8.通过脚本安装 keystone 服务

iaas-install-keystone.sh

9.通过脚本安装 glance 服务

iaas-install-glance.sh

10.通过脚本安装 nova 服务

iaas-install-nova-controller.sh

iaas-install-nova-compute.sh

11.通过脚本安装 neutron 服务

iaas-install-neutron-controller.sh

iaas-install-neutron-compute.sh

12.通过脚本安装 dashboard 服务

iaas-install-dashboard.sh

使用浏览器访问Dashboard：http://192.168.200.50/dashboard，验证安装结果

13.通过脚本安装 swift 服务，执行过程中需要确认登录controller 节点和输入controller 节点root密码

source /etc/xiandian/openrc.sh

iaas-install-swift-controller.sh

iaas-install-swift-compute.sh

输入密码：000000

compute上安装swift

14.验证swift：

source /etc/keystone/admin-openrc.sh

swift stat

swift list

swift post gw001

swift list

swift upload gw001 anaconda-ks.cfg

swift list gw001

15.通过脚本安装 cinder 服务

iaas-install-cinder-controller.sh

iaas-install-cinder-compute.sh
